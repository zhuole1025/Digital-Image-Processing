#### 击中或击不中变换

![](../pic/击中或击不中变换.png)

击中击不中变换(Hit Miss Transform, HMT), 是通过同时探测图像的内部和外部, 进而获取更多的内外标记, 体现更多信息的一种方法. 他的应用有很多, 
特别是在图像识别以及细化方面.
